Instance Deletion	✅	Compiler sekarang mengenali perintah delete object instance ... dan mengubahnya menjadi kode C# .Delete().

5. Struktur & Lainnya
Komponen	Status	Penjelasan Kode
Operations	❌	Belum ada. Loop JSON kamu hanya membaca states. Tidak ada pembacaan operations di level kelas.
Bridges	✅	IMPLEMENTED. 59 bridges across 8 categories (DateTime, String, Collection, Math, Conversion, Logging, File, Misc). Uses TIM:: prefix for all bridge calls. Sequential resolution with max 10 iterations for nested bridges. Depth-tracked argument parsing to preserve all arguments correctly.
Functions ✅	Compiler sekarang bisa membaca model function global dari JSON, menerjemahkan parameter, menangani tipe data (termasuk perbaikan void tadi), dan mengonversi isi fungsinya menjadi static method C#..
Inter-component Messaging ✅ Regex: Port::Msg(...) to Target diubah menjadi Target.Port.Msg(...), dan Port::Msg(...) (implisit) diubah menjadi this.Port.Msg(...).
Object Action Language for Non-State Actions	❌	Belum ada. Karena parser kamu hanya melihat action_oal di dalam states, OAL di tempat lain (operasi/fungsi) tidak akan terkompilasi.
Date and Time	✅	Parsial. Kamu menggunakan CurrentTime di OAL, tapi compiler tidak mengubahnya menjadi DateTime.Now. C# akan menganggap itu variabel lokal yang mungkin tidak ada.
